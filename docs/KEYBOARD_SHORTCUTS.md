# Keyboard Shortcuts - Quick Reference

## Linux Package Installer

### Main Actions
| Shortcut | Action | Where |
|----------|--------|-------|
| `Ctrl + O` | Open/Browse Package File | Install Tab |
| `Ctrl + I` | Install Selected Package | Install Tab |
| `Ctrl + Q` | Quit Application | Anywhere |

### Navigation
| Shortcut | Action | Where |
|----------|--------|-------|
| `F5` | Refresh History | History Tab |
| `Ctrl + Tab` | Switch to Next Tab | Anywhere |
| `Tab` | Move to Next Element | Anywhere |
| `Shift + Tab` | Move to Previous Element | Anywhere |

### Window Management
| Shortcut | Action | Where |
|----------|--------|-------|
| `Alt + F4` | Minimize to Tray | Anywhere |
| `Enter` | Activate/Click Focused Button | Anywhere |
| `Esc` | Cancel Dialog/Close Popup | Dialogs |
